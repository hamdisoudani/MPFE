# MPFE frontend

Next.js 16 (App Router, React 19, Tailwind v3) studio for live-rendering syllabi
generated by the LangGraph agent in `apps/agent`.

## Architecture

- **SSE stream (`@langchain/langgraph-sdk/react::useStream`)** — rejoins active
  run on reload via `reconnectOnMount: true`. Drives a tiny message pane.
- **Custom events** — `onCustomEvent` feeds `useAgentProgress` reducer.
  Powers phase chips, progress bars, attempt counters. Contract lives in
  `src/lib/types.ts` (`AgentEvent`).
- **Supabase Realtime** — `useSyllabusStore` subscribes to `postgres_changes`
  on `syllabuses / chapters / lessons / activities`. Source of truth for the
  center plan view. Survives reloads because Postgres holds the rows.
- **Interrupt handling** — `ClarifyForm` renders when `stream.interrupt.value.kind === "clarification"`
  and resumes via `stream.submit(undefined, { command: { resume: answers } })`.

## Run

```bash
cp .env.example .env.local     # fill in keys
npm install
npm run dev
```

## Test & build

```bash
npm run test         # vitest (jsdom)
npm run typecheck
npm run build
```

## Deploy on Railway

Uses standalone Next.js output + `Dockerfile`. `railway.json` wires it up;
env vars on Railway must mirror `.env.example`.
